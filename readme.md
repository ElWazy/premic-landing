# Premic

## Como instalar el projecto

### Requisitos
- [node.js](https://nodejs.org/en/) (yo utilizo la v16.16.0)
- npm (viene con node.js)

### Instalacion

```bash
git clone https://github.com/ElWazy/premic.git

cd premic
npm i

cd client
npm i

cd ..
npm run build
npm compile

node ./dist/index.js
```
